# Getting started

#### Requirements

To compile and run this project, you will need:
* the NASM assembler
* QEMU
* the GNU make

#### Running the code

Simply use the following commands in a terminal:
```
git clone https://github.com/AnyKeyShik/Snake-Boot.git
cd Snake-Boot
make run
```
